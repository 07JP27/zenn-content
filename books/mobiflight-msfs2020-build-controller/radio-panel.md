# 本チャプターの内容
最初の一歩として、LEDを使ったシンプルな出力から始めましょう。
このチャプターではパーキングブレーキの状態をLEDで表示することを題材に、シンプルな二値(ON/OFF)の出力を学びます。
パーキングブレーキがONの時はLEDは点灯し、OFFの時は消灯するような動作です。

# 準備するもの
- Arduino
- LED
- 抵抗(330~1kΩ程度)
- ブレッドボード
- ジャンパーワイヤー
