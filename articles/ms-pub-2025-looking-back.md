# 活動サマリー
Microsoft (有志) Publicationは45人のメンバーが参加しており、昨年から微増しました。
そして2025年の投稿記事数は275記事、これは3日に約2記事投稿していることになります。
また、それらの記事に記されている文字数を合計すると2,762,844文字となり、昨年から１万文字増加しました。記事数は大きく差はないため、**1記事あたりの密度が上がっている**ことがわかります。
そして読者のみなさまから頂いたいいね数は**昨年の2.5倍**となる4,002となりました。**読んでいただいた上にさらにリアクションもいただいているみなさま、本当にありがとうございます！**

# タイトル ワードクラウド
**各記事のタイトルから**ワードクラウドを作成しました。「Agent Framework」や「GitHub Copilot」「MCPサーバー」などMicrosoft界隈の今年を表す世相になっていることを感じるとともに、生成AIのトレンドは昨年に引き続き勢いが衰えていないように感じます。。
また昨年同様「Defender」や「Sentinel」、「Entra」もランクインしており、ITの全てにおいて重視されるセキュリティ・ガバナンスをメンバーも注目していることがわかります。

# タグ
各記事に付けられている**タグ**を集計してみました。
やはり我々がメイントピックとして扱っている「Microsoft」や「Azure」が上位に位置しています。特にAzureタグは全記事の約半分の56%に付与されています。
2023年が78%→2024年が65%→2025年が56%となっており、**Azureだけに限らない幅広いMicrosoft製品についての記事が投稿される**Publicationになっています。
その証拠にタグのラインキングには「GitHub」や「.NET」、「Microsoft Sentinel」などAzure以外のタグもランクインしてきています。
また記事投稿コンテスト「Zennfes 2025（zennfes2025infra）」に参加したメンバーもいるようです！
